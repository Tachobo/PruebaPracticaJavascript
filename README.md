Objetivo del proyecto

El objetivo de este proyecto es demostrar la capacidad de implementar soluciones en JavaScript para distintos problemas de programación, utilizando funciones, arreglos, objetos, callbacks y manejo de datos dinámicos.

Instalación

1. Clonar el repositorio:

git clone https://github.com/Tachobo/PruebaPracticaJavascript.git

2. Entrar al directorio del repositorio

cd Disco duro donde está la carpeta/PruebaPracticaJavascript

3. Instalar dependencias (carpeta node_modules)

npm install prompt-sync

Ejecución

Se escribe en consola node y el nombre del archivo, en este caso node app.js esto abre el menu que da una lista a elegir de 17 ejercicios
posteriormente pide que se elija uno de los ejercicios para que se ejecute

Ejemplo: 

Se elige el ejercicio 1
